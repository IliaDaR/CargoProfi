Всё, что нужно перевозчику

Инструменты для владельца парка и водителей в одной системе

GPS-трекинг рейсов

Водитель нажимает «Начать рейс» — координаты пишутся каждую минуту. Пробег считается автоматически по формуле гаверсинусов.

Путевые листы

PDF по форме Приказа Минтранса. Дата, госномер, водитель, пробег, время выезда/возврата — подставляются автоматически из рейса.

Учёт расходов

Топливо, стоянка, ремонт — с фото чеков и GPS-меткой. Сводная таблица по категориям для владельца.

Расчёт зарплаты

Правило: % от дохода рейса или фикс за рейс. Расчёт за любой период, ведомость и история выплат.

Контроль автопарка

Все машины на дашборде. Статус: в рейсе или свободен. История по каждому автомобилю.

Безопасность

Ролевой доступ: водитель видит свои рейсы, владелец — весь парк. Данные хранятся в Firebase с шифрованием.

Как это работает

1

Регистрация

Владелец создаёт аккаунт, добавляет машины и водителей.

→
2

Рейс

Водитель нажимает «Начать рейс» — включается GPS-трекинг.

→
3

Учёт

Расходы фиксируются с фото чеков. Пробег считается автоматически.

→
4

Результат

Владелец получает путевой лист, сводку расходов и расчёт зарплаты.

Приложение для водителя

Доступно на Android. GPS-трекинг в фоне, фото чеков, история рейсов. Работает даже без интернета — данные синхронизируются при подключении.
